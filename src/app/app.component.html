EzTable

Dynamic Table Component
1. EzTable
Use <eztable [data]="myJSONArray"> </eztable> to create bootstrap table and provide (array of json data) data to automatically create the headers and rows.
2. Table with custom headers
use headers with key-value pair to provide the custom headers. Keys should be same as of keys of row's data.
3. EzTable - Striped Rows
Use class table-striped to add zebra-striping to the table row within the <tbody>.
4. Condensed Table
use table-sm to make tables more compact by cutting cell padding in half.
5. Hoverable Table
Use class table-hover to enable a hover state on table rows within a <tbody>.
6. Bordered Table
use table-bordered to make tables more compact by cutting cell padding in half.
7. Always responsive
Use table-responsive class for horizontally scrolling tables.
8. EzTable - Sortable
Use <eztable [data]="myJSONArray"> </eztable> to create bootstrap table and provide data (array of json data) <data> to automatically create the headers and rows.
9. EzTable - Custom Row Component
use headers with key-value pair to provide the custom headers. Keys should be same as of keys of row's data.
